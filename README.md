# Plumage

A chaotic farming, exploration, and chicken-management game built with Godot.

Start with a small farm, raise a growing flock of chickens, gather resources from the world around you, and defend your operation from whatever decides to wander too close. Expand your farm, collect useful materials, experiment with different equipment, and try to keep your feathered empire running.

Plumage combines farming, resource gathering, combat, exploration, and a healthy amount of chicken-related nonsense into one package.

## What Is Plumage?

Plumage started as a farming game and gradually accumulated additional mechanics until it became difficult to describe in a single sentence.

You'll spend time collecting resources, managing chickens, exploring the world, gathering materials, fighting enemies, and improving your farm. Different systems interact with one another, allowing you to grow your operation while unlocking new possibilities for progression.

The overall goal is simple:

Get more chickens.

Everything else is just a step toward achieving that objective.

## Development

### Requirements

- Godot 4.4 or newer

### Opening the Project (if you're a nerd)

1. Clone the repository:

```bash
git clone https://github.com/ArandomPsi/farmingsim.git
```

2. Open Godot.
3. Import the project using `project.godot`.
4. Run the main scene.

If you don't know what any of that means, you probably don't need the source code.

### Why Is The Source Public?

Because open source projects are cool, and because sometimes people like looking at how things are put together.

Feel free to poke around, learn from it, break it, fix it, or contribute improvements.
