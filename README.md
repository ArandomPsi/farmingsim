# Plumage

A chaotic farming and chicken-management game built with Godot.

Raise chickens, gather resources, fight monsters, expand your farm, and try to keep your feathered empire alive.

## Play

Play the latest version on itch.io:

https://arandompsi.itch.io/plumage

## Features

- Chicken farming and management
- Resource gathering and mining
- Inventory system
- Multiple weapons
- Monster encounters
- Building and rope-placement mechanics
- Top-down action gameplay
- Persistent farm progression

## Controls

| Action | Input |
|----------|----------|
| Move | WASD |
| Interact | E |
| Shoot / Use Weapon | Left Mouse Button |
| Place Rope | Right Mouse Button |
| Reload | R |
| Weapon Slot 1 | 1 |
| Weapon Slot 2 | 2 |
| Weapon Slot 3 | 3 |
| Cycle Weapons | Mouse Wheel |
| Open Inventory | Tab / I |
| exit menu | Escape |

## Development

### Requirements

- Godot 4.4 or newer

### Opening the Project (if you're a nerd)

1. Clone the repository:

```bash
git clone https://github.com/ArandomPsi/farmingsim.git
```

2. Open Godot.
3. Import the project using `project.godot`.
4. Run the main scene.
